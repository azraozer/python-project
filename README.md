## Yapay Zeka Önerilerini Kapatma

Workspace seviyesinde AI önerilerini kapatmak için `.vscode/settings.json` zaten eklendi.

Eğer uzantıları tamamen kaldırmak isterseniz, VS Code'un `code` CLI'si yüklüyse aşağıdaki betiği çalıştırabilirsiniz:

```bash
./scripts/disable_ai_extensions.sh
```

Betik `github.copilot`, `vscodeintellicode`, `TabNine` gibi yaygın AI uzantılarını kaldırmayı dener.
